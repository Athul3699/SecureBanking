# Requirements for backend app
Python 3.6+
pip3

## Packages to be installed
`Flask==1.1.1`

## How to run
for linux/mac users

`cd backend/api`
`export FLASK_APP=driver.py`
`flask run`

for windows users

`cd backend/api`
`set FLASK_APP=driver.py`
`flask run`
